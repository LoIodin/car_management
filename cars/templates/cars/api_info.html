API документация

API эндпоинты

  • GET /api/cars/ - получение списка автомобилей.
  • GET /api/cars/<id>/ - получение информации о конкретном автомобиле.
  • POST /api/cars/ - создание нового автомобиля.
  • PUT /api/cars/<id>/ - обновление информации о автомобиле. Нужно быть автором записи об автомобиле.
  • DELETE /api/cars/<id>/ - удаление автомобиля. Нужно быть автором записи об автомобиле.
  • GET /api/cars/<id>/comments/ - получение комментариев к автомобилю.
  • POST /api/cars/<id>/comments/ - добавление нового комментария к автомобилю. Требуется авторизация.

Как получить токен

Чтобы использовать API, вам нужен токен аутентификации. Следуйте следующим шагам, чтобы получить токен:

  1. Зарегистрируйтесь или зайдите в свой аккаунт.
  2. Отправьте POST-запрос на адрес /api/token/ со своим логином и паролем:
    3. curl -X POST http://127.0.0.1:8000/api/token/ \
-H "Content-Type: application/json" \
-d '{"username": "Ваш_логин", "password": "Ваш_пароль"}'
  3. Вы получите токен в ответе. Используйте этот токен в заголовке Authorization при отправке запросов к API:
    4. -H "Authorization: Token ваш_токен"
